Warum GitHub?

Server als Host für git Repository Möglichkeit public repository von anderen zu explorieren. kann dadurch lernen, zusammen arbeiten. besserer Ort zum interagieren statt einer Email. pull request oder issue kann gestellt werden. Verschiedene Collaborator an einer Arbeit. kostenlos für Studierende und Mitarbeitende in der Wissenschaft

Arbeit ist weiterhin lokal (local), aber Repository wird auch remote (also online) zur Verfügung gestellt. Weitere Features wie Project Management und Discussions

Vorteile gegenüber Dropboxen Versionskontrolle nicht über solch lange Zeit und umsonst möglich Immer nur für einen File im Projekt dort einsehbar Kombination von beiden kann gerne mal zu Fehlern führen
